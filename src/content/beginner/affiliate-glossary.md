## 为什么需要了解术语？

在进入 Affiliate Marketing 的世界后，你会遇到大量的术语和缩写。如果不理解这些术语，看教程、读 Case Study、甚至和 AM（Affiliate Manager）沟通都会很困难。

这篇文章整理了最常用的 50+ 个术语，按类别分类。

## 基础概念

| 术语 | 全称 | 中文 | 解释 |
|------|------|------|------|
| **AM** | Affiliate Marketing | 联盟行销 | 按效果付费的在线营销模式 |
| **Affiliate** | - | 联盟客 | 推广产品赚取佣金的人（你） |
| **Advertiser** | - | 广告主 | 拥有产品/服务的一方 |
| **Network** | Affiliate Network | 联盟网络 | 连接广告主和联盟客的平台 |
| **AM** | Affiliate Manager | 联盟经理 | 联盟网络中对接 Affiliate 的人 |

## 广告平台

| 术语 | 全称 | 解释 |
|------|------|------|
| **FB** | Facebook | Facebook 广告平台 |
| **IG** | Instagram | Instagram 广告 |
| **GAds** | Google Ads | Google 广告平台 |
| **GDN** | Google Display Network | Google 展示广告网络 |
| **Pmax** | Performance Max | Google 的全渠道智能广告 |
| **TikTok** | TikTok Ads | TikTok 广告平台 |
| **Native** | Native Ads | 原生广告（如 Taboola、Outbrain） |

## 计费模式

| 术语 | 全称 | 中文 | 解释 |
|------|------|------|------|
| **CPM** | Cost Per Mille | 千次展示成本 | 每展示1000次的费用 |
| **CPC** | Cost Per Click | 每次点击成本 | 每次点击的费用 |
| **CPA** | Cost Per Action | 每次行动成本 | 每次转化（注册/购买）的费用 |
| **CPL** | Cost Per Lead | 每条线索成本 | 获取一个潜在客户的成本 |
| **CPS** | Cost Per Sale | 每次销售成本 | 按销售额比例支付 |
| **RevShare** | Revenue Share | 收入分成 | 按用户持续消费分成 |
| **CPI** | Cost Per Install | 每次安装成本 | 按 APP 安装付费 |

## 核心指标

| 术语 | 全称 | 解释 |
|------|------|------|
| **CTR** | Click Through Rate | 点击率 = 点击数 / 展示数 × 100% |
| **CR** | Conversion Rate | 转化率 = 转化数 / 点击数 × 100% |
| **EPC** | Earnings Per Click | 每点击收益 = 总收入 / 总点击数 |
| **ROI** | Return On Investment | 投资回报率 = (收入 - 成本) / 成本 × 100% |
| **ROAS** | Return On Ad Spend | 广告支出回报率 = 收入 / 广告成本 |
| **eCPM** | Effective CPM | 有效千次展示收益 |
| **CVR** | Conversion Rate | 同 CR，转化率 |

## 流量相关

| 术语 | 解释 |
|------|------|
| **GEO** | 目标国家/地区 |
| **Tier 1** | 一线国家（美国、英国、加拿大等） |
| **Tier 2** | 二线国家（巴西、墨西哥、波兰等） |
| **Tier 3** | 三线国家（印度、印尼、菲律宾等） |
| **Targeting** | 定向，选择目标受众 |
| **Creative** | 广告创意（图片/视频+文案） |
| **Ad Copy** | 广告文案 |

## Landing Page 相关

| 术语 | 全称 | 解释 |
|------|------|------|
| **LP** | Landing Page | 落地页 |
| **Pre-lander** | - | 预着陆页，在用户到达 Offer 页面前的过渡页 |
| **Bridge Page** | - | 同 Pre-lander |
| **Lander** | - | Landing Page 的简称 |
| **A/B Testing** | - | A/B 测试，对比两个版本的效果 |
| **Multivariate** | - | 多变量测试 |

## 追踪相关

| 术语 | 解释 |
|------|------|
| **Tracker** | 追踪器（Bemob、Voluum 等） |
| **Click ID** | 点击唯一标识符 |
| **Postback** | 转化回传，通知追踪器转化发生 |
| **Pixel** | 追踪像素 |
| **S2S** | Server to Server，服务端到服务端的追踪方式 |
| **Cookies** | 浏览器 Cookie，用于追踪用户行为 |

## Offer 相关

| 术语 | 解释 |
|------|------|
| **SOI** | Single Opt-In，单步注册（只需提交一次） |
| **DOI** | Double Opt-In，双重确认注册 |
| **CC Submit** | 信用卡提交 |
| **Payout** | 佣金/报酬 |
| **Cap** | 投放上限（每日/每月最多转化数） |
| **Flow** | Offer 的转化流程 |
| **Sweepstakes** | 抽奖类 Offer |
| **Nutra** | 健康/保健品 Offer |
| **VOD** | Video On Demand，视频点播类 Offer |
| **Dating SOI** | 约会类单步注册 Offer |

## 其他常见术语

| 术语 | 解释 |
|------|------|
| **Campaign** | 广告活动/广告系列 |
| **Ad Set** | 广告组（Facebook 的层级概念） |
| **Ad** | 单条广告 |
| **Budget** | 预算 |
| **Daily Budget** | 每日预算 |
| **Lifetime Budget** | 总预算 |
| **Scaling** | 扩大盈利 Campaign 的规模 |
| **Vertical** | 垂直领域/品类 |
| **Spy Tool** | 竞品监测工具，用于查看对手的广告 |
| **Cloaking** | 隐藏/伪装技术（注意合规性） |
