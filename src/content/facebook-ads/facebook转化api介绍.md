Facebook转化API（Conversion API，缩写是CAPI）对于我们来说，就是把转化传给FB的一个工具。通常来说，我们的转化是联盟确认已经产生后，把转化传给跟踪软件，跟踪软件再通过转化API传给FB，那么我们就能在FB上看到转化了。



那么转化API我们在哪里取呢？首先，在FB的事件管理器中，我们找到“settings”。





![](/images/640-1-.png)

下滑到Conversion API





![](/images/640.png)

点击“Generate access toke”，就会生成一段口令，这个就是我们的转化API。





![](/images/640-2-.png)

如果你使用的跟踪软件提供了FB集成功能，那你可以在配置页面里把生成的口令直接复制进去。





![](/images/640-3-.png)

这样转化产生后，就可以通过跟踪软件回传给Facebook了。





![](/images/640-4-.png)
